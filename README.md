本專案源於對財經知識的興趣，旨在透過 Python 自動化技術，將繁瑣的股價搜集與基本面分析流程簡化。使用者只需輸入股票代碼，程式即可自動抓取最新的市場數據，並產出格式精美、具備「台股慣用配色（漲紅跌綠）」的 Excel 分析報表。

核心功能
多市場支援：支援台股 (TW) 及美股 (US) 數據抓取。
自動化報表：自動生成包含「歷史數據」與「公司概況」雙分頁的 Excel 檔案。
視覺化標記：自動計算每日漲跌幅。
漲跌配色：漲幅以紅色顯示，跌幅以綠色顯示（符合台灣市場習慣）。
基本面快照：彙整產業類別、當前現值、市值及本益比 (P/E Ratio) 等關鍵指標。
防封鎖機制：內建禮貌性延遲，確保程式運作的穩定性。

技術棧
資料來源：yfinance (Yahoo Finance API)
數據處理：Pandas
報表輸出：Openpyxl (Excel 引擎)
樣式設計：Pandas Styler
